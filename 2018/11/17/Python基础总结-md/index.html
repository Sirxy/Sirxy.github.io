Python基础总结.md

一、注释
1、 注释的排错性
先注释一部分代码，然后执行另外一部分代码，查看报错，用于调试。
2、注释的注意点
多行注释的嵌套：
如果外面是三个单引号，则里面要用三个双引号；如果外面是三个双引号，则里面要用三个单引号。

二、变量
1、概念：可以改变的量，具体是指内存中的一段存储空间。
2、变量的三种声明方式：
a = 1
b = 2

a, b = 1, 2

a = b = 3 (注意：这种方式的id(a) == id(b))

3、变量的命名规范：
字母数字下划线，首字母不能为数字。
严格区分大小写，且不能使用关键字。
变量命名有意义，且不能使用中文哦。
import keyword
print(keyword.kwlist) # 打印出33个关键字
4、变量的交换
a = 3
b = 10
a, b = b, a
print(a, b) # 10 3

三、Python的六大数据类型
包括：Number型、字符串型、列表型、元组型、集合型、字典型。其中后五种是容器类型数据，Number型又包括int型、float型、bool型、complex型。
